
# TheFuture ⏳

TheFuture is a simple Time Capsule–style website where users can store messages for their future selves and unlock them at a later time.

Live Website:
[https://aadidandekar.github.io/TheFuture/](https://aadidandekar.github.io/TheFuture/)

## Features

* Clean and simple user interface
* Frontend-only project using HTML, CSS, and JavaScript
* Lightweight and fast
* Hosted using GitHub Pages

## Built With

* HTML
* CSS
* JavaScript

## Project Structure

TheFuture/
index.html
style.css
app.js

## Deployment

This website is deployed using GitHub Pages.
Any changes pushed to the main branch are automatically reflected on the live site.

## Purpose

This project was built as a learning and experimental project to explore frontend web development, GitHub Pages deployment, and creative ideas like digital time capsules.
